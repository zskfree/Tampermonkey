# 强密码生成器

简洁的 Tampermonkey 用户脚本，用于在注册 / 表单页面生成**高强度密码**并提供一键复制与可选注入功能。

---

## 主要功能 ✅

- 一键生成高强度随机密码（可配置长度与字符集）
- 单击复制到剪贴板，Shift+点击或启用“自动注入”可直接填入表单
- 拖拽调整悬浮按钮位置并持久化保存
- 设置面板可配置偏好（长度、字符类型、排除相似字符等）
- Tampermonkey 菜单支持白名单与位置重置等快捷操作

---

## 安装与使用 🔧

1. 安装 Tampermonkey（或兼容的用户脚本管理器）。
2. 将 `强密码生成器.js` 添加为新脚本并保存。
3. 打开包含密码字段或注册页面，页面右下会出现悬浮按钮 `密`：
   - 单击：生成并复制密码
   - Shift + 单击：生成并尝试注入到当前输入框
   - 双击：打开设置面板
   - 拖拽：移动按钮位置（会保存）

---

## 配置项 ✨

- 密码长度：4–128（默认 16）
- 字符集：小写 / 大写 / 数字 / 符号
- 排除相似字符：如 `0`, `O`, `l`, `1`
- 显示策略：在所有站点显示 / 仅在含密码字段时显示 / 使用白名单

设置通过 Tampermonkey 存储持久化（GM_setValue / GM_getValue）。

---

## 权限与隐私 🔒

脚本会使用以下权限：

- `GM_setValue`, `GM_getValue` — 保存偏好
- `GM_setClipboard` — 在必要时写入剪贴板
- `GM_registerMenuCommand` — 注册菜单命令

脚本在本地生成、复制与注入密码，不会将任何密码或表单数据发送到网络。

---

## 常见问题 ❓

- 按钮未出现：可在 Tampermonkey 菜单中启用“在所有站点显示”或将当前站点加入白名单；脚本也会在页面动态加载密码字段时尝试显示按钮。
- 复制失败：脚本优先使用 `navigator.clipboard`，若不可用会回退到 `GM_setClipboard` 或传统 textarea 复制方法。
