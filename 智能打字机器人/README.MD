# 智能打字机器人

智能打字机器人是一个Tampermonkey浏览器脚本，可以模拟人工逐字输入文本内容。该脚本具有友好的用户界面，支持多种文本输入方式和速度调节。

## 功能特性

- **逐字模拟输入**：模拟真实用户的打字行为，包括按键事件和输入延迟
- **直观UI界面**：使用Shadow DOM创建独立的用户界面，不影响页面原有样式
- **元素拾取**：可选择页面上的输入框、文本域或可编辑元素
- **速度调节**：提供5档速度调节，从极慢到极快
- **暂停/继续/停止**：完全控制打字过程
- **文本导入**：支持粘贴、拖拽文件或手动输入文本
- **进度显示**：实时显示打字进度和状态
- **便捷操作**：支持拖拽移动面板，最小化显示

## 安装方法

1. 确保已安装Tampermonkey浏览器扩展
2. 将 `智能打字机器人.js` 文件内容复制到新的Tampermonkey脚本中
3. 保存并启用脚本

## 使用方法

1. **输入文本**：在面板的文本框中输入或粘贴需要模拟输入的内容
2. **选择目标**：点击"选择输入框"按钮，然后点击页面上想要输入的目标元素
3. **调节速度**：通过滑块选择合适的打字速度
4. **开始输入**：点击"开始打字"按钮启动模拟输入
5. **控制过程**：使用暂停/停止按钮控制输入过程

## 快捷键

- `ESC`：退出元素拾取模式
- `Ctrl+Shift+T`：显示/隐藏控制面板

## 支持的元素类型

- `<input>` 标签（文本框、文本域等）
- `<textarea>` 标签
- `contenteditable` 属性为 true 的元素

## 注意事项

- 脚本在所有网站上都可以运行（`*://*/*`）
- 所有操作都在当前页面进行，不会发送数据到外部服务器
- 脚本使用原生DOM API，不依赖外部库
