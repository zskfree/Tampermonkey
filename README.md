# Tampermonkey 实用脚本集合

本项目是由本人（[@zskfree](https://github.com/zskfree)）开发和维护的油猴（Tampermonkey）实用脚本插件合集，全部脚本均为 JavaScript 编写。项目收录了多个针对不同网站和实际场景的用户脚本，旨在提升日常网页浏览和操作体验。

## 项目特色

- 多种实用脚本涵盖常见网站增强、自动化操作、内容优化等需求
- 所有脚本均可直接导入 Tampermonkey 插件使用
- 持续更新与维护，欢迎建议和反馈

## 快速开始

1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 浏览器扩展
2. 浏览本项目，选择你需要的脚本
3. 打开脚本内容，复制粘贴到 Tampermonkey 新建脚本页面并保存即可生效

## 常见脚本类型

- 广告屏蔽、页面美化
- 自动签到、批量自动操作
- 快捷功能增强
- 针对特定站点的专用工具
